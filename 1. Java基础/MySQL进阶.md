# 事务

## 执行流程

1. 关闭自动提交
1. 开启一个事务
1. 提交 / 回滚
1. 开启自动提交

```mysql
SET autocommit = 0;  -- 1. 关闭自动提交

START TRANSACTION;  -- 2. 开启一个事务
-- 具体执行语句

COMMIT;  -- 3. 提交(提交后不可撤销[持久化])
ROLLBACK; -- / 回滚(回到事务开启前的状态)
-- 结束该事务

SET autocommit = 0;  -- 4. 开启自动提交
```

# 或者 ===================
mysql -u用户名 -p密码 库名

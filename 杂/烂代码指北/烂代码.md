# 命名

## 错误拼写

> 将大量的单词拼写成自己设计后的组合, *如 getUserName  -> gatUsarNane*



## 易混字符

> 在少量标识符插入易混字符, *如 int insertVal  -> int ínsərtVal*



## 相似命名

> 少量的相似变量名, *如 swimmer = swimner + 1;*
